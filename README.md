# GstTIOVX

GstTIOVX is a project that provides a framework for integrating the
Texas Instruments implementation of OpenVX standard into GStreamer.

## Installing GstTIOVX

```
meson build --prefix=/usr
ninja -C build/ -j8
ninja -C build/ install
```
